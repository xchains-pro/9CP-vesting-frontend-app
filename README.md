# Token Vesting Frontend App for XChains
## Project setup
### Build tools
- npm
- yarn

### Dev tools and framework
- Next.js
- Wagmi
- Viem
- ConnectKit
- Tailwind & ShadCn

### 3rd-Party We3 services
- Node provider: Alchemy
- Wallet connect


## Run the app
### Run in dev mode 
cp .env.example .env.local
yarn dev

### Build a production version
yarn build
yarn start
